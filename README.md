## Install

Add the tap:

```bash
brew tap bearcove/tap
```

Then install a formula:

```bash
brew install bearcove/tap/home
```

## Uninstall

Uninstall using homebrew:

```bash
brew uninstall bearcove/tap/home
brew untap bearcove/tap
```
